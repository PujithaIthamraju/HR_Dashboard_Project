# HR Analytics Dashboard (Advanced Excel)

## Project Overview
This **HR Analytics Dashboard** provides insights into employee attrition, workforce demographics, and job satisfaction. It helps HR teams make data-driven decisions to improve employee retention and performance.


## Business Objectives
- Monitor employee attrition rate
- Analyze attrition by job role, department, and education
- Track workforce demographics
- Identify age groups with higher attrition


## Key Metrics (KPIs)
- **Total Employees:** 1470  
- **Active Employees:** 1233  
- **Employees Left (Attrition):** 237  
- **Attrition Rate:** 16.12%  
- **Average Employee Age:** 37  


## Dashboard Components
### Employee Count by Gender
- Gender-wise workforce distribution using visual indicators

### Attrition by Job Role
- Horizontal bar chart showing attrition across roles such as:
  - Sales Executive
  - Research Scientist
  - Laboratory Technician
  - Human Resources

### Age Group-wise Attrition
- Column chart analyzing attrition across age groups

### Department-wise Attrition
- Pie chart highlighting departments contributing most to attrition

### Attrition by Education
- Bar chart showing attrition based on education level


## Interactive Filters
- Education Field slicer
- Dynamic dashboard interaction


## Tools & Techniques Used
- Microsoft Excel
- Pivot Tables
- Advanced Charts
- Slicers
- HR Metrics Calculation
- Dashboard UI Design


## Key Insights
- Certain job roles have significantly higher attrition
- Younger age groups show higher employee turnover
- Education level influences attrition trends
- HR and Sales departments contribute more to attrition
